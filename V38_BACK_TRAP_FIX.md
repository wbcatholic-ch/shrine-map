# V38 뒤로가기 종료 기준 통일

## 수정 내용
- 매일미사·주요기도문·성가 팝업/외부 이동 후 커버로 돌아와도 커버에서는 항상 뒤로가기 2번으로 앱 종료되도록 통일했습니다.
- 커버 복귀 시 history trap을 공통 함수 `oaiEnsureCoverBackTrap()`으로 재확립합니다.
- 매일미사·기도문·성가 빠른 팝업이 열린 상태에서 뒤로가기를 누르면 앱 종료가 아니라 팝업을 먼저 닫습니다.
- 다른 카테고리의 기존 뒤로가기 흐름은 유지했습니다.

## 확인할 것
1. 성지/성당/피정의집 진입 후 커버 복귀 → 뒤로가기 1회 토스트, 2회 종료
2. 매일미사/기도문/성가 팝업 진입 후 커버 복귀 → 뒤로가기 1회 토스트, 2회 종료
3. 팝업이 열린 상태에서 뒤로가기 → 팝업 닫힘
4. 기도문 본문에서는 뒤로가기 → 목록, 다시 뒤로가기 → 커버
